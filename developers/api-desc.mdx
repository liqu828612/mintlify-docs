# 接口说明
* 更新时间：2022-01-16 21:31:02

## 通讯协议
商户接入PayerMax 服务，调用API须遵循以下规则：

| 类型	      | 说明 |
|-----------|-----|
| 传输方式	  | 为保证交易安全性，采用HTTPS传输，且TLS版本不低于1.2，详情参考 TLS升级说明  |
| 提交方式	| 采用POST方法提交  | 
| 数据格式	 | 提交和返回数据都为application/json格式 |
| 字符编码	| 统一采用UTF-8字符编码 |
| 签名算法	| SHA256WithRSA | 
| 签名要求	| 请求和接收数据均需要校验签名，详细方法请参考 配置与签名 | 
| 判断逻辑	| 先判断协议字段返回，再判断业务返回，最后判断交易状态 |
